Crear una clase FileSystemService.

Responsabilidad:
Aplicar cambios en archivos del sistema según FileChange.

Requisitos:
1. Método principal:
   void applyChanges(List<FileChange> changes)

2. Soportar acciones:
   - CREATE → crea archivo (y carpetas si no existen)
   - MODIFY → sobrescribe archivo existente

3. Usar java.nio.file:
   - Files.createDirectories
   - Files.writeString

4. Manejar errores de forma simple (try/catch y log)

5. No agregar complejidad innecesaria

Importante:
- Mantener código simple
- No usar librerías externas
